## ?? QUICK START (3 Steps)

### Step 1: Load Data
```cpp
MNISTLoader loader(28, 28);
loader.loadTrainingData("D:/Project/Neural_Net_Graph/input/trainingSample");
```

### Step 2: Prepare Data
```cpp
loader.shuffle();
loader.normalize();

std::vector<MNISTLoader::Sample> train, test;
loader.splitData(0.8, train, test);

std::vector<Eigen::VectorXd> x_train, y_train;
loader.convertToModelInput(train, x_train, y_train);
```

### Step 3: Train Model
```cpp
sequential model({
    {784},
    {256, activation::RELU},
    {128, activation::RELU},
    {10, activation::SOFTMAX}
});

model.fit(x_train, y_train, 100, 32);
```
